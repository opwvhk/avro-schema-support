Detects the optional type syntax (new in Avro 1.11.1), offering to replace with the corresponding union. Useful if you're using an older version of Avro.

Example:

record MyData {
    string? name = "Anonymous";
    string? description = null;
}


	The quick fix replaces this shorthand syntax with unions.




	Result after applying to both fields:



record MyData {
    union {string, null} name = "Anonymous";
    union {null, string} description = null;
}
