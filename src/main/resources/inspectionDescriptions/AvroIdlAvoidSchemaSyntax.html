Detects the schema syntax (since Avro 1.12.0), and offering to replace it with a protocol.

Useful if you're using an older version of Avro.

Example:

namespace my.name.space;
schema array<Message>

record Message {
	string title;
	string content = "";
}

enum Status {
	ON, OFF, UNKNOWN
} = UNKNOWN;

The quick fix wraps the file content with a protocol, adding a namespace annotation if appropriate. It drops the schema declaration.

Result after applying the fix: 

@namespace("my.name.space")
protocol MessageLess {
	record Message {
		string title;
		string content = "";
	}

	enum Status {
		ON, OFF, UNKNOWN
	} = UNKNOWN;
}
