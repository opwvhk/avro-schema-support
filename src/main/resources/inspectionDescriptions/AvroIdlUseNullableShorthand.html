Identifies unions that can be simplified using the new syntax in Avro 1.11.1.

Example:

record MyData {
    union { string, null } name = "Anonymous";
    union { null, string } description = null;
}

The quick fix replaces these unions with the new shorthand syntax.

Result after applying both: 

record MyData {
    string? name = "Anonymous";
    string? description = null;
}
